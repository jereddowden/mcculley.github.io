macOS/iOS Keyboard Shortcuts

Typesetting

control ⌘ + superscript (e.g., x2)
control ⌘ - subscript (e.g., H2O)

Miscellaneous Symbols

option 4 ¢ (cent)
option shift 8 ° (degree)
alt 2 ™ (trademark)

Screenshots

add control to save to the clipboard instead of the desktop

⌘ shift 3 a file containing a screenshot of the entire screen is added to your desktop
⌘ shift 4 a file containing a screenshot of an area is added to your desktop
⌘ shift 4 space a file containing a screenshot of a specified window is added to your desktop

Text Navigation/Selection

add shift to select text between cursor and jump point

⌘ ← jump to beginning of line
⌘ → jump to end of line
option ← jump to beginning of current word
option → jump to end of current word
option ↑ jump to beginning of document
option ↓ jump to end of document
